## WebSocket API

`cargo_kind` can be: `butterfly`/`leaf`/`cloud`/`drop`

```json
{
  "data": {
    "kind": "cargo",
    "cargo_kind": "butterfly",
    "id": 1,
    "directory": "/path/to/texture/file"
  }
}
```

```json
{
  "data": {
    "kind": "launch",
    "cargo_amount": 20
  }
}
```

```json
{
  "data": {
    "kind": "weather",
    "raining": true
  }
}
```

```json
{
  "data": {
    "kind": "population",
    "amount": 100
  }
}
```
